jGlideMenu

Version 0.6.7

jGlideMenu is an updated version of the fastFind Menu that supports inline and AJAX menu definitions and uses an easier syntax than the original version.

Better documentation will be provided with future versions.

Notes

This version is now compatable with all recent versions of jQuery and jQuery UI, including jQuery 2.x and jQuery 3.x.
Click Here for details on how to use this plugin with jQuery 2.x.
